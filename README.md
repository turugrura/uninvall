# Descriptions

- Playground for NestJS (NodeJS framework)

# Goals

- Serve Web API
  - RESTful
  - GraphQL
- Implement OOP Design patterns
  - Adapter
  - Strategy
  - ...

# Features

- Authentication by JWT
  - /login API & /me
  - support Role
- Store data
  - by MongoDB
- CRUD RESTful API
  - /cats
  - /friend-translator
- Adaptable logger
- etc
  - Nest scheduler
